Random Task Generator
**Автор:** [Тимур Халилов]

Описание
Приложение на Python с графическим интерфейсом (Tkinter), которое помогает побороть прокрастинацию, выбирая случайную задачу из списка.

Функционал
- Генерация случайной задачи.
- Фильтрация истории по категориям (Учёба, Спорт, Работа).
- Автоматическое сохранение истории в JSON файл.

Как запустить
1. Установите Python 3.x.
2. Скачайте `main.py`.
3. Запустите через терминал: `python main.py`.

Примеры тестов
1. Нажать "Сгенерировать" — в списке должна появиться новая запись.
2. Выбрать фильтр "Спорт" — в списке должны остаться только спортивные задачи.
3. Закрыть и открыть приложение — история должна сохраниться.
